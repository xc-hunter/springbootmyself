# Springfox使用
* 引入依赖 implementation group: 'io.springfox', name: 'springfox-boot-starter', version: '3.0.0'
* 配置Swagger配置类(未研究spring security情况下如何使用)
* 创建controler,启动
* 访问path/swagger-ui/index.html like http://localhost:8084/swagger-ui/index.html
